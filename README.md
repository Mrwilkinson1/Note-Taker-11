# Note Taker 11

## Description
This Note taking app was created to save notes to a database.
Type, save, and delete notes as needed.

## Installation
Npm and Node.js are required. In the terminal run:
```python

npm init -y

npm i express

npm i uuid

nodemon server.js
```
